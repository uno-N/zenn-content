## 3プラットフォームの使い分け

| | Note | Zenn | Qiita |
|---|---|---|---|
| 読者 | 医療者・一般・初心者 | エンジニア・個人開発者 | エンジニア（最大規模） |
| 内容 | 体験・感情・ストーリー | 学習記録・個人開発体験 | エラー解決・手順まとめ |
| 文体 | 会話・共感重視 | 体験ベースの技術記事 | 検索・解決重視 |
| 収益化 | 有料記事・サポート | 本・有料記事 | なし |
| 拡散力 | フォロワー重視 | 中規模 | 大規模・検索流入 |

## 記事の横展開イメージ

```
同じ開発体験
  ├── Note    → 感情・失敗談・ストーリー（共感を得る）
  ├── Zenn    → 学びの記録・体験ベースの技術解説
  └── Qiita   → 手順・エラー解決（検索で来る人向け）
```

1つの体験から3本書けるので、ネタを無駄にしない。
